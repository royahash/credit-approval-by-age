# Credit Card Approval Rates by Age Group

## Project Description

This project is a narrative-focused D3 visualization that explores approval patterns in a credit card applications dataset. The visualization uses a non-interactive chart to show how credit card approval rates differ across age groups.

The main takeaway is that younger applicants in this dataset had lower approval rates, while approval rates increased across older age groups. Since the dataset is anonymized, the visualization does not claim why this pattern exists. Instead, it focuses on clearly communicating the approval-rate difference shown in the data.

## Narrative Focus

The chart highlights a steady increase in approval rates from the youngest age group to the oldest age group. The lollipop chart was chosen because it makes the comparison between age groups easy to read while keeping the design clean and public-facing.

## Data Cleaning

The original dataset was cleaned and grouped into age categories. Approval and denial counts were calculated for each age group, then converted into approval and denial percentages.
